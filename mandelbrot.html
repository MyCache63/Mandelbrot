Mouse Controls

  • Scroll Zoom in/out toward cursor
  • Drag Draw rectangle, release to zoom into that area
  • Shift+Drag Pan the view
  • Double-click Zoom in 3x at that point
  • Right-click Zoom out 3x

Keyboard Shortcuts

  • H Toggle this help
  • P Toggle control panel
  • C Toggle color cycling
  • R Reset to home view
  • B Go back in history
  • S Save PNG screenshot
  • + / - Increase / decrease iterations
  • ←→↑↓ Pan the view
  • [ / ] Shift palette offset
  • 1-8 Switch fractal type

Tips

  • Increase iterations when zoomed deep for more detail.
  • Try color cycling with different palettes for amazing animations.
  • In Julia mode, enable "Click Mandelbrot to pick c" for interactive Julia exploration.
